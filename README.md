Bem-vindo(a) ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar a minha trajetória profissional, os meus projetos e as minhas competências como Desenvolvedora FullStack.

## 📌 Sobre o Projeto
Este portfólio foi construído do zero com foco num design moderno, interativo e totalmente responsivo. A interface utiliza os conceitos de Glassmorphism (efeito de vidro translúcido) e elementos 3D iterativos, garantindo uma experiência de utilizador (UX) premium e fluida.

## 🚀 Funcionalidades
* **Efeito Glassmorphism:** Cards e secções com fundos translúcidos e desfoque (blur) para um aspeto moderno.
* **Scroll Reveal:** Animações suaves em que os elementos surgem no ecrã à medida que o utilizador faz scroll.
* **3D Tilt Effect:** Os cards da secção de portfólio interagem com o movimento do rato, criando um efeito de perspetiva 3D dinâmico utilizando Vanilla JavaScript.
* **Cards Interativos (Flip 3D):** A secção de trajetória académica e profissional possui cards que "viram" a 180 graus ao clicar, revelando mais informações.
* **Design Responsivo:** O layout adapta-se perfeitamente a dispositivos móveis, tablets e desktops.

## 🛠️ Tecnologias Utilizadas no Portfólio
Este projeto front-end foi desenvolvido utilizando apenas tecnologias nativas da web, sem frameworks pesados, para garantir a máxima performance:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada, CSS Grid/Flexbox, animações @keyframes e variáveis de cor.
